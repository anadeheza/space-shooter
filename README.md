#Space shooter
2D arcade game made with HTML, CSS, and JavaScript. 
The game puts you in command of a starship tasked with defending the galaxy. The game features mouse-based controls, where your ship automatically fires at the enemies while you navigate the screen, all while tracking your health and score in real-time. The game includes shortcuts accessible via keyboard: press "M" to access the main menu, "I" to view instructions, and "P" to toggle the pause state.
